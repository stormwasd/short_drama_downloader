# 使用说明

## 安装步骤

1. 确保已安装 Python 3.7 或更高版本
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
3. 运行程序：
   ```bash
   python main.py
   ```

## 功能说明

### 新建任务

1. **任务名称**：为下载任务起一个名称，方便管理
2. **来源**：选择短剧来源平台（shortlinetv 或 reelshort）
3. **剧集名称**：输入短剧的名称
4. **剧集网址**：
   - shortlinetv: 输入类似 `https://shortlinetv.com/videos/440` 的网址
   - reelshort: 输入类似 `https://www.reelshort.com/episodes/trailer-you-fired-a-fashion-icon-687f2a41314aed63020928f9-dr1wo1epdw?play_time=1` 的网址
5. **剧集区间**：
   - 可以选择下载的剧集范围（0-200）
   - 如果两个值都是0，则下载所有剧集
   - shortlinetv 默认从第1集开始
   - reelshort 默认从第0集开始（但实际会跳过预告片）
6. **存储地址**：选择下载文件的保存位置

### 任务进度

- **下载中**：查看正在下载的剧集，可以：
  - 查看实时下载进度
  - 勾选未开始下载的剧集进行删除
- **已完成**：查看已完成的下载任务

## 注意事项

1. 下载过程中请保持网络连接
2. 确保有足够的磁盘空间
3. 如果下载失败，可以在"下载中"标签页查看错误信息
4. 数据会自动保存到本地数据库，关闭程序后再次打开仍能看到之前的任务

## 技术特性

- 支持多任务并发下载（默认3个并发）
- 实时进度显示
- 数据持久化存储
- 错误处理和重试机制
- 美观的图形界面
